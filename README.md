# movie-night
To setup Server
```
clone project
cd movie-night-server
npm install
```
To setup Client
```
cd movie-night-app
npm install
```

To setup Project
```
clone project
npm install
```

To run server
```
cd movie-night
cd movie-night-server
node app.js
```

To run client
```
cd movie-night
cd movie-night-app
yarn start
(agree to run other local host if running locally)
```
